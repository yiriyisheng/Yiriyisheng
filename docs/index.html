在日常里，慢慢生长出力量

我不是专家，也不是导师。
只是一个认真生活的人。

在音乐里安顿情绪，
在饮食与作息中照顾身体，
在阳台的一小片土地上，
学习耐心与时间。

我把这些记录下来，
给同样在努力生活的你。

🎵 音乐

音乐不是表演，是陪伴。

🌱 养生

把身体，当作一位需要被理解的朋友。

🪴 成长

成长，其实是在重新认识时间。

🧭 人生

普通人的生活，也值得被认真书写。

如果你最近有点累，
不用急着改变什么。

在这里慢慢看，
也许你会发现，
你已经在往前走了。
